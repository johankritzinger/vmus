About

This app is developed in free time as an open source project. Anybody interested in contributing, see https://github.com/johankritzinger/vmus

Recent changes

  • Check when record is complete enough to submit
  • Allow submission of records
  • Show User id in records page if logged in

Known issues

  • When the record is complete enough to submit, you have to save, then go back in before the submit button is available
  • When submitting a record, it is still there, but hidden, with no interface to see or delete submitted records yet
  • No proper checking of status of submission.
  • Instability - if it crashes, just try again and close errors ;)
  • Many others, see and add issues at the github link

Next steps
